NetOffice Developer Toolbox

The NetOffice Developer Toolbox is a Productivity Tool for .NET developers who create solutions with and for Microsoft Office.

Feature
Office Compatibility Check your .NET assemblies for compatibility against several MS-Office versions
Application Observer Monitor running MS-Office applications
Registry Editor Clone of the Windows-registry editor for the MS-Office Hive Keys
Addin Guard Monitor the LoadBehaviour of COMAddins
Outlook Security Supress the Outlook Security Dialog
Office UI Find and delete dead UI Items
Project Wizard Project assistant for VS(Express Solutions)
